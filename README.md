# MarketShot

MarketShot is a modern marketplace platform built with Django.

## Features

- Multi-store support
- Product management
- Categories and attributes
- Store dashboard
- Order management
- Responsive design

## Tech Stack

- Django
- SQLite/PostgreSQL
- Tailwind CSS
- JavaScript

## Installation

```bash
git clone https://github.com/USERNAME/marketshot.git
cd marketshot

python -m venv venv
source venv/bin/activate

pip install -r requirements.txt

python manage.py migrate
python manage.py runserver
